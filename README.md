Tested to run on OS X:

* 10.14 Mojave
* 10.13 High Sierra
* 10.12 Sierra
* 10.11 El Capitan
* 10.10 Yosemite

## Getting Started 

__Liftoff:__

Make this script accessible to all users by placing it in: `/usr/local/bin`; 

set the executable bit on the file:

`chmod +x about_dis_mac.sh`;   

then call it:

`bash about_dis_mac`.
